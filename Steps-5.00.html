Troubles that occurred during the project 5

1: In One of my data set that has the information of the Health Care provider and the respective health rating scores, the start date and end date was mixed as DateRange for some of the files and for others it was given as Start Date and End Date. On top of that I had multiple ranged for one record which made it even more complicated so I had to first load them as is to the staging area under the StartDate column but right after they are loaded I set up a stored procedure that will be executed by an execute sql task to clean this records and split the date ranges to the appropriate StartDate and EndDate Column.
2: The other major problem I have faced was that my Score column had mix of null, string and numeric data types so I had to fix this issue before loading  my main staging database since I needed it to be numeric. I also needed to do a lookup on the second data set that I had that had the information about the hospitals employee and available bed count. In doing this I was able to separate the hospitals that reported their employee and bed counts to the ones that didn't and gave the one that didn't 'N/A' value for that columns
3: Since I did not want to load Null values for my score to my ODS as it is my source of my dim tables which then feed my fact tables I had to use and Update Statement to change the Null values to '-1' after being loaded.
